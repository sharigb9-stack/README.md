Description of Cine Plus - Sharks

The proyect involves designing a digital system for selling movie tickets, intended to facilitate ticket purchases, the selection of concessions, and the generation of a final purchase summary.

The main objective is to simulate the operation of a real system like those used in modern cinema box offices or mobile applications.

They system consist of several windows that guide the user step by step from login to purchase confirmation.

This project allows for an understanding of how a graphical interface is structured, how information is distributed across each screen, and how each screen connects to guide the user throughout the entire process.

SCREEN.
Screen 1 login

On this screen, the client finds the main access point to the system. Here, they must enter their username and password in the corresponding fields.

The "Enter" and "Exit" buttons allow them to continue to the system or close the application. The design includes elements alluding to film, such as an image. This screen functions as a security filter that controls which users can access the system.

Screen 2 ticket sales

This window displays the list of available movies along with their showtimes.

The user can select their preferred movie and choose their desired showtime. Ticket prices are also shown: $90 for adults and $80 for children, along with a field that calculates the subtotal based on the selected amounts.

The "Next" button takes the user to the concession stand, while "Back" returns them to the homepage. In the center is a bitmap-style box, which represents the image or poster of the selected movie.

Screen 3 sweet shop

In this section, customers can add extra items such as soft drinks, popcron, and special combo meals.

Each item includes its price and a space to select the desired quantity. The subtotal is automatically calculated based on the selected items.

This screen helps complete the shopping experience, allowing customers to purchase food items before proceding to final payment. The "Back" and "Next" buttons allow you to navigate between purchasing tickets and the checkout stage.

Screen 4 payment of the purchase

This is the final screen of the process. Here you'll find a complete summary of everything the customer selected: -Movie -Ticket type and quantity -Showtime -Concessions -Ticket total -Concessions total -Grand total It also includes a thank you message for the purchase.

The customer can correct their purchase by going back using the "Back" button, or finalize the transaction by pressing "Confirm".
